bash基本命令介绍

bash命令的基本语法

cd

全称： change directory
功能： 改变当前所在的目录

~： 指的其实就是当前用户的用户根目录
等价于： /c/users/当前用户的用户名/

..: 指的就是上级目录

.: 指的就是当前目录

pwd

全称：Print Work Directory
功能：打印当前所在的目录！

ls

全称：list
功能：展示当前目录下所有的内容信息

有参数的情况
功能： 不切换文件夹，就直接展示指定的文件夹下的所有的文件信息

有选项的情况
-a 选项的意思是 all 也就是将所有的内容全部展示出来，包括隐藏文件
-l 选项的意思是 list 就是将所有的内容的详细信息以表格的形式显示出来

touch

功能：创建文件

mkdir

make directory
功能：创建文件夹

rm

remove
功能： 删除文件
注意： 这个删除不会进回收站

rm命令也可以用来删除文件夹，但是需要加上一个选项 -r
rm -r 文件夹名称

rm -f 可以用来强制删除

rmdir

remove directory
功能： 删除文件夹
注意： 这个删除不会进回收站
如果文件夹中有内容，则无法删除！！

cp

copy
功能： 复制文件 或者 文件夹

要复制文件夹 ，需要加上选项 -r

mv

move
移动文件 文件夹
